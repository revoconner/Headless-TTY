# Security Policy

- Latest version will get a year of support for security updates.
- LTS releases will get security updates for 5 years but no platform change support.
- Security updates will be released based on vulnerability discoveries. 


## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.5.0   | [YES] Latest- Till December 10th, 2027 |
| 2.0.0  LTS | [YES] LTS - Security updates till December 5th, 2030 |
| < 2.0.0  | [NO] Not supported, do not use|

## Reporting a Vulnerability

You can use the Security tab on the top of this repository to submit private Vulnerability. 
If the Vulnerability is legitimate and can be reproduced at my end, I will implement a fix
as soon as possible.


Fake AI generated reports will result in user being banned.
